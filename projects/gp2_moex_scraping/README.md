# GP2 — MOEX Market Overview (Scraping & API)

**Фокус:** сбор данных (MOEX ISS API + web-scraping), очистка и базовый риск/доходность анализ индексов.  
**Инструменты:** Python, pandas, numpy, requests, BeautifulSoup, matplotlib (+ seaborn для heatmap).

## Что сделано
- Реализована функция постраничной загрузки свечей с MOEX ISS API (поддержка JSON/JSONP).
- Собран единый датасет по набору индексов (акции/облигации/прочее) в длинном формате.
- Подготовлены недельные цены и недельные лог-доходности.
- Посчитаны метрики (как в учебном проекте): годовая доходность, волатильность, Sharpe, Max Drawdown, Calmar, skew/kurtosis, CAGR.
- Построены: нормированная динамика индексов и матрица корреляций недельных доходностей.
- Сформулированы короткие рекомендации (диверсификация, риск-профили).
