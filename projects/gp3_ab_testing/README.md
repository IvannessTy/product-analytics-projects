# GP3 — A/B Test: Landing Page

**Фокус:** подготовка данных A/B-теста, EDA, проверка гипотез, интерпретация результатов.  
**Инструменты:** Python, pandas, numpy, matplotlib, statsmodels.

## Что сделано
- Проверка качества данных: пропуски, дубликаты, mismatch (group vs landing_page).
- Очистка: удаление mismatch-наблюдений, пользователей в обеих группах, дубликатов `user_id`.
- EDA: баланс групп, конверсии, динамика конверсии по дням.
- Статистический тест: z-test для долей (альтернатива — рост конверсии).
- Дополнительно: bootstrap CI для разницы конверсий (эффект в диапазоне).

## Файлы

## Как запустить
1) Установите зависимости: `pip install -r requirements.txt`
